## Telegram-бот для ежедневного отслеживания самочувствия и поиска закономерностей в твоём образе жизни.

## Возможности

- **Запись настроения** по шкале 1–5 (инлайн-кнопки)
- **Учёт рабочих часов** (быстрые кнопки или ввод вручную)
- **Учёт сна** (быстрые кнопки или ввод вручную)
- **Опциональный комментарий** к дню
- **Статистика за неделю / месяц** с текстовой сводкой
- **Инсайты** — анализ влияния сна и работы на настроение
- **Графики** (через matplotlib)
- **Настройка времени напоминаний**
- **Очистка данных** с подтверждением


## База данных


Используется SQLite (файл `mood_tracker.db` создаётся автоматически).

## Таблицы

`users`
| Поле | Тип | Описание |
|------|-----|----------|
| `user_id` | INTEGER PK | Telegram ID |

| `username` | TEXT | @username |

| `first_name` | TEXT | Имя |

| `created_at` | TEXT | Дата регистрации |

| `reminder_time` | TEXT | Время напоминания |


`daily_entries`

| Поле | Тип | Описание |

|------|-----|----------|

| `id` | INTEGER PK | Автоинкремент |

| `user_id` | INTEGER FK | Ссылка на users |

| `entry_date` | TEXT | Дата записи |

| `mood` | INTEGER | Настроение 1–5 |

| `work_hours` | REAL | Часы работы |

| `sleep_hours` | REAL | Часы сна |

| `comment` | TEXT | Комментарий |
